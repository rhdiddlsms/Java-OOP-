# 오버로딩

오버로딩이란 하나의 클래스 내에 인수의 개수나 형이 다른 동일한 이름의 메서드를 여러 개 정의하는 것이다.

```java 
// 오버로딩 : 클래스에 같은 이름의 메서드가 여러 개 존재하는것.
class Calc {
	int add(int a, int b) {
		return a + b;
	}
	
	int add(int a) {
		return a + 1;
	}
	
	double add(double a, double b) {
		return a + b;
	}
}

public class Calculation {

	public static void main(String[] args) {
		//객체 생성
		Calc calc = new Calc();
		//메서드 호출
		int nReturn1 = calc.add(3, 9);
		int nReturn2 = calc.add(5);
		double nReturn3 = calc.add(3.6, 9.5);
		
		System.out.println(nReturn1+":"+ nReturn2+":" + nReturn3);
		
	}

}
```
